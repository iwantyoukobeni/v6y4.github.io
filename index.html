Hentaiv1

API Documentation

this api allows u to get hentai anime girls

Making a Request


    import requests

    
    response = requests.get('https://hentaiapiv1.pythonanywhere.com/random_hentai_image')

    
    if response.status_code == 200:
        // Parse the JSON response
        data = response.json();
        // Print the image URL
        console.log(data['image_url']);
    else:
        console.log(`Error: ${response.status_code}`);

Animepicsv1

API Documentation

this api allows you to get pics of anime

Making a Request


    import requests

response = requests.get('https://animepicsv1.pythonanywhere.com/random_anime_image')

if response.status_code == 200:
    data = response.json()
    image_url = data.get('image_url')
    if image_url:
        print(f'Random anime Image: {image_url}')
    else:
        print('Image URL not found in response')
else:
    print(f'Error: {response.status_code}')

End of API Documentation animepicsv1
